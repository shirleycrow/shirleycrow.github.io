发表于 2021-05-19 更新于 2021-07-13 分类于 Notes 阅读次数：

好不容易搭好了博客，在此记录一下踩的各种坑和经验。

有关部署

本地测非常重要

部署到github上面前一定要本地测过，不然不能更新到主页上。
也就是：
hexo clean //清除本地public缓存
hexo g //重新编译public
hexo s //在http://localhost:4000/预览本地修改
hexo d //第三步确认无误后，部署到github
这样就能确保本地的设置信息可以同步到主页上~

报错“Unable to build page”

The page build failed for the “master” branch with the following error:
Unable to build page. Please try again later.
For information on troubleshooting Jekyll see:
“https://docs.github.com/articles/troubleshooting-jekyll-builds"

gitpage默认是用Jekyll来编写的，但是我用的是Hexo来建站。Hexo有一些Jekyll不支持的文件命名方式。
比如开头是下划线。Hexo最重要的配置文件_config.yml就是以下划线开头的
这时候要把Jekyll强行扭到Hexo。
添加一个.nojekyll的空文件到根目录即可解决。

1.Unity直接以Android平台打开工程（不用手动切换平台）

找到相应的Unity.exe所在的目录
F:\Unity5.5.5\Unity\Editor\Unity.exe

在目录下打开命令行窗口
执行：

Unity.exe -projectPath E:\Dev\Client -buildTarget android

2.Unity中的 Api Compatibility Level

PlayerSettings里面
.NET 4.x 比 .NET Standard 2.0 要高
支持的方法也 更多
例如：

using System.Runtime.Remoting;

这个在 .NET Standard 2.0里就没有
切换成 .NET 4.x就支持了
